# Cheaper Nexus

Professional Digital Marketing & Creative Agency Landing Page.

## Features
- Multi-language support (English, Chinese, Malay)
- Responsive Design
- Services Showcase (Video Production, Social Media, Google & SEO, etc.)
- Contact Form
- Automated Deployment via GitHub Actions

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion (Motion)
- Vite

## Deployment

### GitHub Pages
This project is configured with GitHub Actions. Simply push to the `main` branch to deploy.

### Firebase Hosting
1. Install Firebase CLI: `npm install -g firebase-tools`
2. Run `npm run deploy:firebase`
